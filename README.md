# Game On - Plataforma de Videojuegos

Proyecto personal del Técnico en Desarrollo de Software (SENA):  
Una página web responsiva (mobile-first) dedicada a videojuegos clásicos y modernos. Incluye página principal + páginas individuales con info detallada de cada juego (resumen, requisitos, precio en Steam, etc.).

## Tecnologías usadas
- HTML5 (estructura semántica)
- CSS3 (Flexbox/Grid, media queries, diseño adaptable)
- Imágenes locales y enlaces externos a Steam

## Cómo verlo
1. Clona el repo:  
   `git clone https://github.com/kevinvillamil1607-rgb/Game-on-.git`
2. Abre `index.html` en tu navegador.
